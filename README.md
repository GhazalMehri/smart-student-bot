📚 ربات همراه دانشجو (Student Assistant Bot)
📌 توضیحات پروژه
ربات همراه دانشجو یک ربات تلگرامی است که برای کمک به دانشجویان در مدیریت کارهای درسی، برنامه‌ریزی و یادآوری وظایف طراحی شده است. این پروژه با استفاده از زبان Python و کتابخانه‌ی python-telegram-bot ساخته شده است.
✨ قابلیت‌ها
اضافه کردن تکالیف و وظایف درسی
مشاهده لیست کارهای ثبت شده
حذف وظایف انجام شده
سیستم یادآوری (Reminders)
مدیریت ساده و سریع برنامه‌های درسی
🛠 تکنولوژی‌ها
Python 🐍
python-telegram-bot
Git & GitHub
🚀 نحوه اجرا
1. نصب پیش‌نیازها
pip install -r requirements.txt 
2. اجرای پروژه
python main.py 
📁 ساختار پروژه
bot/ ├── main.py ├── handlers.py ├── database.py ├── config.py requirements.txt README.md 
👨‍💻 توسعه‌دهنده
دانشجوی مهندسی کامپیوتر
